# Project Status Dashboard

The dashboard must report operational truth, not aspiration.

Required fields:

- server on/off
- scheduler running/not_running
- LLM workers: none
- autonomous agents: none
- local scripts: available
- beta-check status
- smoke browser status
- persistent storage status
- GitHub remote status
- Telegram configured/not_configured/out_of_scope
- staging URL configured/missing
- critical blockers

Forbidden language:

- "autonomous agents active"
- "beta stable" without smoke, health and durable storage
- "staging ready" without deployment URL and remote baseline
