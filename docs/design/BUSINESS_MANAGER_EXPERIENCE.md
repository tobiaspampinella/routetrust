# Business Manager Experience Checklist

## Must Be Understandable

- operation status
- active routes
- active drivers
- delayed routes
- failed deliveries
- open incidents
- SLA at risk
- bug reports
- system status

## Minimum KPI Set

- active routes
- completed deliveries
- failed deliveries
- delayed routes
- open incidents
- SLA at risk
- active drivers
- bug reports
- system status

## Checklist

- dashboard hierarchy is obvious
- metrics do not compete visually
- incidents are distinguishable from bugs
- system status does not overstate reality
- project status page is operationally honest
- CMS and settings pages are not dead ends
