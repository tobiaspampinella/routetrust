# Brand Guidelines

## Idea de Marca

RouteTrust es un producto de inteligencia operacional. El tono debe sentirse controlado, visual y confiable.

## Mantener

- lenguaje directo
- sustantivos operativos
- estado de producto honesto
- marco de supervision humana

## Evitar

- hype futurista
- teatro de autonomia
- ruido decorativo de dashboard
- gestos de app de consumo

## Direccion Visual

- base dark graphite
- acento azul operacional fuerte
- uso medido de verde, amber y rojo
- superficies de metricas con alto contraste
- gradientes ornamentales minimos
