# CFO Audit

Generated: 2026-06-01
Role: CFO Strategy Agent

## Verdict

The current cost base is low because the product is local/demo. Real SaaS costs begin once DB, staging, logs, support and onboarding exist. Pricing cannot be finalized until the unit of value is chosen.

## Pricing Hypotheses

Potential pricing units:

- per active driver/month
- per tracked delivery
- per tenant/company/month
- per operations module
- implementation fee plus SaaS subscription

Recommended beta package:

- setup fee for controlled implementation
- monthly platform fee
- capped driver/delivery volume
- no usage-based complexity until telemetry is real

## Cost Drivers

- managed Postgres
- app hosting
- map provider
- notifications
- logs/monitoring
- support hours
- implementation work
- QA/security

## Financial Risks

- Map/traffic APIs can destroy margin if included without caps.
- Manual implementation can consume all revenue if not productized.
- Support burden will rise sharply if customer tracking is sold before backend maturity.

## Required CFO Controls

- Cost per tenant.
- Cost per active driver.
- Cost per 1,000 tracked deliveries.
- Support hours per tenant.
- Gross margin model before pilot pricing.
